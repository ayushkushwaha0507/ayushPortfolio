# ayushPortfolio
Portfolio Website

This is a fully responsive portfolio website built using HTML, CSS, and JavaScript without any frameworks. It features smooth animations powered by GSAP and a Google Sheets integration for handling form submissions.

Features:

Responsive Design: Works seamlessly on all devices.

GSAP Animations: Smooth and modern animations for a dynamic user experience.

Google Sheets Form Integration: Contact form submissions are stored directly in Google Sheets.

Interactive Elements: Engaging and interactive user interface.
